## 🆘 **Common Issues & Fixes**

| Problem | Solution |
|---------|----------|
| `npm: command not found` | Restart computer, then try again |
| Port 5000 in use | Edit `.env`, change PORT to 5001 |
| Installation hangs | Press Ctrl+C, delete `node_modules`, try again |
| Script won't run | Right-click → "Run with PowerShell" |
